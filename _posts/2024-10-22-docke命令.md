#### 查看具体镜像和容器占用的空间

```shell
# 查看所有镜像的大小
docker images

# 查看所有容器的大小
docker ps -a --size
```

#### 查看 Docker 系统信息

```shell
docker system df
```

这个命令会显示：

* 镜像的总数和占用的空间
* 容器的总数和占用的空间
* 卷的总数和占用的空间


### 清理构建缓存

```shell
docker builder prune
```
