    ros2开发环境搭建

    March 15, 2025

    最近想做些 ros2 相关的开发。记录一下本地配置的过程。 因为想开发调试 rclcpp 等中间件层的代码，而不只是做上层应用开发，所以选择了 build from source。

    个人开发习惯是 docker + 本地文件夹映射进 docker。

    repo 拉取代码 #

    官方默认是用 vcs 拉的代码，比较习惯 repo 工具管理多仓库，改了一版 repo manifest.xml：https://github.com/Manchey/ros2-repo-manifest/tree/master

    repo 改用清华源 # 

    1

    export REPO_URL='https://mirrors.tuna.tsinghua.edu.cn/git/git-repo'

# 或 socks5 代理
export http_proxy=socks5://127.0.0.1:你的代理端口

export http_proxy=socks5://127.0.0.1:7891
export https_proxy=socks5://127.0.0.1:7891
