    bazel编译go项目

    November 26, 2024
    Tech/Bazel, Tech/Go, Blog

    bazel编译项目 #

    Go相关代码，改bazel编译 #

    gazelle生成各个BUILD.bazel #

    添加gazelle #

    • gazelle是用来自动生成go项目bazel BUILD的工具。

      • deps.bzl：

            http_archive(
        name = "bazel_gazelle",
        integrity = "sha256-12v3pg/YsFBEQJDfooN6Tq+YKeEWVhjuNdzspcvfWNU=",
        urls = [
            "https://mirror.bazel.build/github.com/bazelbuild/bazel-gazelle/releases/download/v0.37.0/bazel-gazelle-v0.37.0.tar.gz",
            "https://github.com/bazelbuild/bazel-gazelle/releases/download/v0.37.0/bazel-gazelle-v0.37.0.tar.gz",
        ],
    )
        • 总的BUILD中添加：
        load("@bazel_gazelle//:def.bzl", "gazelle")

# gazelle:prefix code.deeproute.ai/cloud-infra/c01-fota
gazelle(name = "gazelle")

    添加高版本buildtools #

    • gazelle和bazelbuild/buildtools有冲突，也是版本问题，依赖的buildtools比较新，已有的旧。加了一个新的。

      • https://github.com/bazelbuild/buildtools/blame/main/build/BUILD.bazel#L11C14-L11C14

    运行，生成各个go项目的BUILD # 

    bazel run //:gazelle
